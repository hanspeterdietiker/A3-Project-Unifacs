## 💻 Api do Restaurante :
- A aplicação deve conter um sistema de busca de ferramentas de software utilizando filtros.
- No sistema as ferramentas devem ter uma descrição geral e o link para documentação.
- O sistema precisa poder realizar buscas através de filtros com o tipo de ferramenta.
